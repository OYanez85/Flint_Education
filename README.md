# IoT Bootcamp – Flynt Education

This repository contains all the materials created and used to deliver the **IoT Bootcamp at Flynt Education** for students aged 16–18.

## 📚 Course Overview

The bootcamp introduces students to core concepts in:

- Internet of Things (IoT)
- Programming fundamentals
- Physical computing
- Cybersecurity basics
- AI awareness

The course is structured as a **hands-on, project-based learning experience**.

---

## 🗂️ Repository Structure

### Day 0 – IoT Introduction
- Introduction to IoT concepts
- Slides and handouts

### Day 1 – Scratch
- Visual programming using Scratch
- Games and interactive projects

### Day 2 – Arduino
- Physical computing with Arduino
- Circuits and hardware interaction
- Example projects and libraries

### Day 3 – Raspberry Pi
- Introduction to Linux and Raspberry Pi
- CLI basics and system interaction

### Python Games
- Fun CLI-based Python games for learning programming:
  - Number Guessing Game
  - Rock Paper Scissors
  - Password Strength Checker
  - Quiz Game
  - Typing Speed Challenge
  - And more

---

## 🎯 Learning Objectives

By the end of the bootcamp, students should:

- Understand how software interacts with hardware
- Write basic programs in Python and Scratch
- Use Arduino for simple electronics projects
- Navigate Linux using the command line
- Develop problem-solving and computational thinking skills
